### Development:  
1. Clone the repository  
```bash
git clone https://github.com/Z1xus/smoothie-blender
```
2. Install yarn (if you dont have it)  
```bash
npm install --global yarn
yarn set version latest
```
3. Install dependencies  
```bash
yarn install
```
4. Run it locally  
```bash
yarn dev
```
